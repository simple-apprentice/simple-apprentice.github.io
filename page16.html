Mechanical Hand

This project is my undergraduate graduation project. In this system, 5000 gesture images with different backgrounds, skin colors, and illumination were collected and labeled with labelimg. The self-made data set was divided into a training, verification, and test set. The YOLOv5 model was used to predict five gestures on the test set. Experimental results showed that 16 samples were trained each time. After 100 rounds of training, the model performance index is 81.1% precision, 81.4% recall, and 0.531 MAP.

Based on the original YOLOv5 model, attenuation learning rate and data enhancement methods are further introduced in this paper. The CBAM attention mechanism, multi-scale detection method, and Gaussian filtering are improved to improve the model's efficiency further. The improved model was tested on the same test set, and the experimental results showed that the model performance indexes were 87.2% precision, 84.1% recall, and 0.564 MAP. Through optimization, model accuracy improved by 6.1%, recall rate increased by 2.7%, and MAP improved by 3.2%. Experimental tests show that the proposed enhanced method is feasible and can accurately recognize five basic gestures and further improve the performance of the model.

Finally, the optimized model was deployed on the raspberry PI 4B development board, and the LFD-01 anti-blocking steering gear was used to build the bionic manipulator. Finally, the manipulator reproduction function was realized.
